## ğŸ› ï¸ Installation & Customization

### 1. Simple Installation

Grab the pre-compiled `Unity.theme.css` from the `Theme/` folder and drop it into your BetterDiscord (or Vencord/Aliucord) themes folder.

### 2. Building from Source

If you want to customize the colors, gaps, or fonts:

1.  Navigate to the `Theme/` directory:
    ```bash
    cd Theme
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Customize your variables in `Theme/src/theme/_root.scss`.
4.  Rebuild the theme:
    ```bash
    npm run build
    ```
